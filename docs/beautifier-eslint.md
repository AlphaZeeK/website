## Usage
Below are example configuration files for each of the supported languages.
<details><summary>JSX</summary>
A `.unibeautifyrc.json` file would look like the following:
```json
{
  "JSX": {
    "beautifiers": [
      "ESLint"
    ]
  }
}
```
</details>
<details><summary>JavaScript</summary>
A `.unibeautifyrc.json` file would look like the following:
```json
{
  "JavaScript": {
    "beautifiers": [
      "ESLint"
    ]
  }
}
```
</details>
