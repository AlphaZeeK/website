## Usage
Below are example configuration files for each of the supported languages.
<details><summary>CSS</summary>
A `.unibeautifyrc.json` file would look like the following:
```json
{
  "CSS": {
    "beautifiers": [
      "Prettier"
    ]
  }
}
```
</details>
