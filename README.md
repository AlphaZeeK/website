# Unibeautify Website

## Contributing

Install dependencies:

```
# yarn scripts
npm install

# For Docusaurus
cd website
yarn install
```

Generate the docs for options, languages, and beautifiers:

```
yarn docs
```

We use [Docusaurus](https://docusaurus.io/) to build the website:

```
cd website
yarn start
# navigate to http://localhost:3000
```
